# B1P18-R1-SEM2-2025-2029

Java practice assignments for SEM2.

## UC1: Display Hello World

Run with Maven:

```powershell
mvn compile
mvn exec:java
```

Or run directly with Java:

```powershell
javac -d target/classes src/main/java/HelloApp.java
java -cp target/classes HelloApp
```

Expected output:

```text
Hello World
```
